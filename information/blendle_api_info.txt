article construction logic:
we have an BlendleItem

	Content body:
		BlendleItem._embedded.content.body = array of text content item
			this order is depending on the images
			type = kicker, hl1, intro, byline, p, etc.
			content = the content itself, with html tags in it
	
	Images:
		BlendleItem._embedded.content._embedded.mediasets
			position = position in whole document, this is 0 based, so after body item 22 this image will appear
			type = mostly image
			_embedded = image download information
				BlendleItem._embedded.content._embedded.mediasets._embedded
					original - large - medium - small
						large._links.file = url direct image
						large.width
						large.height
